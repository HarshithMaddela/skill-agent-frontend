# 💻 Skill Agent Frontend (React)

## 📌 Overview

This is the frontend of the AI Skill Assessment Agent.

It provides a clean and interactive interface where users can:

* Upload Resume and Job Description
* View skill evaluation results
* Explore personalized learning plans

---

## ⚙️ Tech Stack

* React.js
* HTML5 + CSS3
* JavaScript
* Axios (API calls)
* React Icons

---

## 🚀 Features

* Resume & Job Description upload interface
* Skill-wise evaluation display
* Gap analysis visualization
* Learning roadmap UI
* Responsive and modern design

---

## 🔗 Backend API

Connected to:
👉 https://github.com/HarshithMaddela/skill-agent-backend

---

## ▶️ Running Locally

```bash
git clone https://github.com/HarshithMaddela/skill-agent-frontend
cd skill-agent-frontend
npm install
npm start
```

App runs on:
👉 http://localhost:3000
